Welcome to the ParaTreeT Github pages

Links to the library code and wiki are listed here.

ParaTreeT/dev

This is the main development repository of ParaTreeT. It currently contains tree abstractions and example programs using them. Documentation can be found on the wiki.

ParaTreeT/dist

This repository contains the distributed version of ParaTreeT written in Charm++. It will be merged with ParaTreeT/dev once the implementation's performance is optimized.

ParaTreeT/cluster-finding

This repository contains the distributed cluster finding library that uses union-find operations. It will be merged with ParaTreeT/dev when it is stabilized and integrated into ChaNGa for galaxy finding.
